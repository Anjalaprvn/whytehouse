# Invoice List - Quick Icon Reference

## Action Icons

| Icon | Color | Name | Function |
|------|-------|------|----------|
| 📱 | 🟢 Green (#25D366) | WhatsApp | Contact customer via WhatsApp |
| ✉️ | 🔵 Blue (#3b82f6) | Email | Send email to customer |
| 👁️ | 🟣 Purple (#8b5cf6) | View | View invoice details |
| ✏️ | ⚫ Black (#111) | Edit | Edit invoice |
| 🗑️ | 🔴 Red (#ef4444) | Delete | Delete invoice |

## Pending Amount Badge

**Color**: 🔴 Red
- Background: Light red (#fee2e2)
- Text: Dark red (#dc2626)
- Bold font
- Rounded corners

**Example**:
```
INV-202601-0001
Pending: ₹5,000  ← RED BADGE
```
